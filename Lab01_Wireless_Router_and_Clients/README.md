# Lab 01 — Configure a Wireless Router and Clients

**Platform:** Cisco Packet Tracer
**Course:** Networking Basics — Cisco NetAcad
**Status:** ✅ Complete

## What This Lab Covers
- Connecting coaxial cables from ISP splitter to modem and TV
- Connecting wired PCs to a home wireless router via Ethernet
- Configuring router DHCP settings and limiting max users
- Changing default admin credentials
- Setting up a 2.4GHz wireless network with custom SSID
- Enabling WPA2 Personal security with a passphrase
- Configuring client devices for DHCP and verifying connectivity

## Key Insight
DHCP is a two-sided protocol. The router being configured as a DHCP
server is not enough — each client device must also be set to DHCP
mode before it will request an address. A common real-world
troubleshooting point.
